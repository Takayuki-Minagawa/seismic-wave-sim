# 地震波と地球内部構造 シミュレーター AI 指示書

## 0. 共通事項

本仕様書は、既存の参考プロジェクト群と同じ設計思想・運用前提で動作する Web 教材の作成指示書である。個別機能に入る前に、全テーマ共通の前提を以下にまとめる。以降の章は、この共通事項を前提として本テーマ固有の要件のみを記述する。

### 0.1 参考プロジェクト

- [GEO-LEARNER](https://takayuki-minagawa.github.io/GEO-LEARNER/) — 観測視点切替 + 学習モジュールの三層構成
- [weather-front-sim](https://takayuki-minagawa.github.io/weather-front-sim/) — 時系列操作 + 観測値グラフ + 学習パネル + クイズ
- [web-planetarium](https://takayuki-minagawa.github.io/web-planetarium/) — 地点・時刻に応じた星空描画と i18n / テーマ切替

これらと同じ設計思想（操作パネル＋可視化ビュー＋学習パネル＋クイズ、i18n、ライト／ダーク切替、静的配信）を踏襲する。

### 0.2 共通仕様

- 静的フロントエンド構成とする（React + TypeScript + Vite を基本、軽量なら Vanilla JS + HTML + CSS も可）
- GitHub Actions により GitHub Pages へデプロイする
- 日本語 UI を基本とし、英語切替（i18n）に対応する
- ライト／ダークテーマ切替に対応する
- PC・タブレット・スマホ対応のレスポンシブ UI にする
- 画面は「操作パネル」「可視化ビュー」「学習パネル（解説＋クイズ）」の三層構成を基本とする
- 教育用途のコンテンツであり、実測・予報・診断など厳密な数値提供を目的にしない
- 公開リポジトリ前提の表現で README と仕様書を書き、個人情報・認証情報・秘密鍵は含めない

### 0.3 共通で再利用したいコンポーネント

- タイムスライダー、再生・一時停止・速度変更コントロール
- 学習パネル・クイズフレーム（状態に応じた解説文生成と正誤判定）
- i18n ロード機構、ライト／ダークテーマ切替
- 3D シーン基盤（Three.js + React Three Fiber）: 3D 系テーマで共用
- 2D グラフ・波形描画基盤（Canvas / D3.js）: 時系列グラフ系テーマで共用
- SVG 編集系 UI（図形選択・ドラッグ・履歴）: 図形操作系テーマで共用

### 0.4 学習パネルの共通方針

- 現在の状態・入力に対する解説を自動生成する
- 観察の着目点や誘導テキストを出す
- クイズモードを併設し、主要な誤解ポイントを問題化する
- 言語切替に追従して文言が再生成される構造にする

## 1. 目的

震源を自由に設定し、P 波・S 波が地球内部をどう伝わり、どこで屈折・反射し、どの地点に到達するかを視覚的に理解できる Web 教材を作成する。

高校地学基礎・地学で扱う「地震波走時曲線」「S 波シャドウゾーン」「地球内部構造」を一体で学べる内容にする。

- P 波は固体・液体を伝わり、S 波は固体のみ伝わる
- 外核が液体であるため S 波シャドウゾーン（103°〜 の範囲）が生まれる
- P 波シャドウゾーンは 103°〜143° 付近
- 走時曲線から各層の速度構造が推定できる

## 2. 開発方針

### 2.1 基本方針

- フロントエンドのみ、静的 Web アプリ
- GitHub Pages 公開
- PC／スマホ対応、日英切替、ライト／ダーク

### 2.2 推奨技術スタック

- React + TypeScript + Vite
- 2D 描画: SVG または Canvas 2D（地球断面の伝播アニメーション）
- グラフ描画: D3.js または軽量 Canvas ライブラリ
- スタイリング: Tailwind CSS または CSS Modules

### 2.3 画面構成

- **地球断面ビュー**: 円形の地球断面。地殻・マントル・外核・内核を色分けし、波の伝播をアニメーション
- **走時曲線パネル**: 震央距離 vs 到達時間のグラフ
- **学習パネル**: 現状解説、クイズ

---

## 3. 必須機能

### 3.1 震源操作

- 地球断面の表面をクリック／ドラッグして震源を設定
- 震源の深さスライダー（0〜700 km、プレート境界地震までをカバー）
- マグニチュード（表示用・波形の強さに影響）

### 3.2 波の伝播アニメーション

- P 波（青）と S 波（赤）を同時に可視化
- 各層で速度が変わることを色の濃淡や速度の変化で表現
- 屈折をスネルの法則相当で近似（層ごとに一定速度の簡易モデル）
- 外核に到達した S 波は吸収され、先へ進まない
- 表面の観測点に到達した瞬間にハイライト＋波形パネルへ記録

### 3.3 観測点

- 地表に観測点を複数配置できる
- 震源との角距離（震央距離）を表示
- 観測点ごとに到達時刻・到達波（P, S, どちらも来ない）を表示
- シャドウゾーンに入る観測点は明確に色分け

### 3.4 走時曲線

- 全観測点の震央距離 vs 到達時刻をプロット
- P 波直達波、PP・PKP などの代表的な波相を色分け（簡易で可）
- S 波の曲線は 103° 付近で切れることを明示
- クリックで観測点と連動ハイライト

### 3.5 地球内部構造パネル

- 地殻・マントル・外核・内核の厚さ・密度・主要ポイントを表示
- ホバーで説明表示
- 各層での P・S 波速度の代表値

### 3.6 学習パネル

- 現在の震源・観測点配置に対する解説
- 「なぜ 103° 以遠で S 波が届かないのか？」などの誘導
- クイズモード
  - シャドウゾーン判定
  - 波の到達順
  - 外核が液体である根拠

- URL クエリで配置共有
- PWA 対応

## 5. データ・計算設計

### 5.1 簡易速度モデル

- 地殻: P 6.0, S 3.5 km/s
- 上部マントル: P 8.0, S 4.5 km/s
- 下部マントル: P 13.0, S 7.0 km/s
- 外核: P 9.0, S 0 km/s（液体）
- 内核: P 11.0, S 3.5 km/s

### 5.2 伝播計算

- 球殻ごとに一定速度とみなし、スネルの法則で屈折経路を離散計算
- 簡易化のためレイトレースで代表ラインを求め、補間描画する
- 到達時間は経路長 / 速度 の積分で算出

### 5.3 走時データ

- プリセット走時曲線は `travelTimes.ts` に事前計算値を格納

---

## 6. アニメーションとビジュアル

- 波の伝播は同心円的な波面と、代表レイの両方を描画
- P 波は青、S 波は赤の半透明ライン
- 外核到達時に S 波を「吸収されるように」フェードアウト
- 観測点でヒットした瞬間に小さなパルスを描画
- 走時曲線は波の到達と同期して点が追加される
