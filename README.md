# 地震波と地球内部構造シミュレーター

P波・S波の地球内部伝播と走時曲線を可視化する教育用Webシミュレーター。

## デモ

https://takayuki-minagawa.github.io/seismic-wave-sim/

## 機能

- **地球断面アニメーション**: P波（青）・S波（赤）の伝播をリアルタイム可視化
- **シャドウゾーン表示**: S波シャドウゾーン（103°以遠）・P波シャドウゾーン（103°〜143°）
- **走時曲線**: 震央距離 vs 到達時間のリアルタイムグラフ（D3.js）
- **観測点管理**: 複数観測点の配置・到達時刻・シャドウゾーン判定
- **学習パネル**: 状態連動解説 + クイズモード（4問）
- **地球内部構造パネル**: 5層モデルの速度・密度情報（ホバー説明）
- 日本語/英語切替・ライト/ダークテーマ・レスポンシブ対応（PC/タブレット/スマホ）

## ローカル開発

```bash
npm install
npm run dev
```

http://localhost:5173/seismic-wave-sim/ にアクセス

## ビルド

```bash
npm run build
```

## 使い方

| 操作 | 動作 |
|---|---|
| 地球をクリック/ドラッグ | 震源を設定（表面クリック→地表震源、内部クリック→深発地震） |
| ダブルクリック | 観測点を追加 |
| ▶ 再生 | 地震波アニメーション開始 |
| 深さスライダー | 震源深さ設定（0〜700 km） |
| プリセット | 実際の大地震データで自動設定 |

## 技術スタック

- React + TypeScript + Vite
- Canvas 2D — 地球断面アニメーション（60fps）
- D3.js — 走時曲線グラフ
- Tailwind CSS v4
- i18next — 日英切替
- GitHub Actions → GitHub Pages

## 速度モデル（簡易）

| 層 | P波 (km/s) | S波 (km/s) |
|---|---|---|
| 地殻 | 6.0 | 3.5 |
| 上部マントル | 8.0 | 4.5 |
| 下部マントル | 13.0 | 7.0 |
| 外核（液体） | 9.0 | — |
| 内核 | 11.0 | 3.5 |
